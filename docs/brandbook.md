## 1. Identidad y Tono

**Marca:** UQOMM  
**Misión:** Proveer sistemas de comunicación subterránea confiables y adaptables para la Minería Digital, garantizando continuidad operativa y seguridad (más de 30 años de experiencia).  
**Tono de Voz:** Técnico-Profesional, Asegurador y Orientado a Resultados. Enfatiza Confianza, Adaptabilidad y Simplicidad.  
**Público Objetivo:** Gerentes de Operaciones, Seguridad, Productividad y Jefes de IT/Telecomunicaciones en la industria minera.  

### Principios de Comunicación
- Claridad técnica sin jerga innecesaria.
- Enfatizar continuidad operativa y seguridad.
- Resaltar experiencia y confiabilidad.
- Evitar tono promocional exagerado.

## 2. Estilo Visual

El estilo visual debe ser potente, de alto contraste y sugerir energía, acción y solidez.

### 2.1 Paleta Principal
| Rol | Color | Hex | Uso |
|-----|-------|-----|-----|
| Primario (Energía/Énfasis) | Rojo/Naranja Fuerte | `#FF4500` | Logo, títulos principales, botones de acción, alertas críticas |
| Fondo Base | Negro / Gris Oscuro | `#1F1F1F` | Fondos generales, bloques principales |
| Texto / Contraste | Blanco | `#FFFFFF` | Texto sobre fondos oscuros |
| Soporte / Elementos Secundarios | Gris Claro | `#AAAAAA` | Líneas divisorias, anotaciones, íconos secundarios |

### 2.2 Colores de Estado (Sugeridos)
| Estado | Color | Hex | Uso |
|--------|-------|-----|-----|
| Éxito | Verde Suave | `#2FAF58` | Confirmaciones, indicadores OK |
| Advertencia | Ámbar | `#FFB020` | Mensajes preventivos |
| Error / Crítico | Rojo/Naranja Primario | `#FF4500` | Alertas críticas |
| Información | Azul Técnico | `#2F80ED` | Mensajes informativos |

> Ajustar si el PDF oficial define variantes exactas.

### 2.3 Tipografía
| Uso | Familia Recomendada | Estilo |
|-----|---------------------|--------|
| Títulos (H1–H2) | Oswald / Impact / Alternate Gothic | Mayúsculas, bold, tracking ligero |
| Subtítulos (H3–H4) | Oswald / Roboto Condensed | Semibold, mayúsculas opcional |
| Cuerpo | Roboto / Arial / Open Sans | Regular o 400–500 |
| Destacados | Roboto Bold | Color primario |
| Códigos / Logs | Consolas / Courier New | Monoespaciado |

### 2.4 Jerarquía Visual
- Títulos en MAYÚSCULAS con alto contraste.
- Separadores sutiles (líneas gris claro #AAAAAA al 40% de opacidad).
- Bloques resaltados con borde o barra lateral en `#FF4500`.
- Espaciado consistente: 8 / 16 / 24 / 40 px (escala base).

### 2.5 Iconografía
- Estilo simple, plano, sin gradientes.
- Color primario o blanco según fondo.
- Trazos consistentes (2–2.5px grosor equivalente).

### 2.6 Imágenes
- Preferir fotografías industriales de minería, equipos y entornos subterráneos.
- Aplicar overlays oscuros (40–60%) para colocar texto legible encima.

## 3. Estructura de Documentos Técnicos

Los documentos deben reflejar un enfoque profesional y de reporte ejecutivo técnico.

### 3.1 Secciones tipo recomendadas
1. Resumen Ejecutivo
2. Objetivo / Alcance
3. Arquitectura / Diseño (si aplica)
4. Componentes / Materiales
5. Flujo Operativo / Casos de Uso
6. Métricas / Riesgos / SLA
7. Conclusiones y Próximos Pasos

### 3.2 Elementos Reutilizables
| Elemento | Formato |
|----------|---------|
| Notas Importantes | Bloque con fondo #1F1F1F, borde izquierdo #FF4500 |
| Advertencia | Fondo oscuro + ícono de alerta + texto en `#FF4500` |
| Tablas | Encabezados con barra inferior en primario |
| Código / Logs | Fondo #121212, texto monoespaciado |
